# وقفة (Waqfa) — محرك الإنذار المبكر من الديون

مساعد مالي ذكي يرصد إشارات الخطر المالي **قبل** وقوعها، خصوصاً مع انتشار خدمات التقسيط
(BNPL — تمارا، تابي)
بين الشباب السعودي. بُني لـ **هاكاثون امد 2026** — مسار الذكاء الاصطناعي التوليدي للتقنية المالية.

> الفكرة: لا نخبرك أين صرفت بعد فوات الأوان — بل نوقفك قبل أن تقع.

## الميزات

- **تحليل ذكي بالـ AI** — تقييم شخصي للوضع المالي عبر نموذج توليدي (Groq / llama-3.3)
- **مناطق إنذار مبكر** — يكتشف الخطر في مرحلة "تحذير" قبل "خطر"
- **توقّعات مالية** — إسقاط مسارك المالي وتنبيه استباقي
- **محاكي "ماذا لو"** — جرّب أثر أي قرار مالي قبل اتخاذه
- **ربط متعدد المصادر** — محاكاة المصرفية المفتوحة (بنك + محفظة + مزوّدي BNPL)
- **ملف تخصيصي** — توصيات تتكيّف مع العمر والدخل والأهداف
- **تعليم مالي تفاعلي** — شرح المصطلحات المالية بالـ AI
- **تقارير ومشاركة** + سجل تحليلات + إشعارات

## التشغيل

```bash
flutter pub get
# انسخ قالب الأسرار وضع مفتاحك
cp lib/services/secrets.dart.example lib/services/secrets.dart
# عدّل secrets.dart وضع مفتاح Groq من https://console.groq.com
flutter run
```

## التقنيات

- **Flutter** (Dart) — واجهة عربية RTL، تصميم داكن، خط Cairo
- **Groq API** (llama-3.3-70b) — التحليل التوليدي مع نظام احتياطي محلي
- **shared_preferences** — تخزين محلي (الملف، السجل)

## الأمان

مفاتيح الـ API في `lib/services/secrets.dart` وهو **مستبعد من git**. لا ترفع مفاتيحك.

## ملاحظة

ربط المصرفية المفتوحة نموذج تجريبي لإثبات المفهوم. التنفيذ الفعلي عبر إطار المصرفية المفتوحة المرخّص من البنك المركزي السعودي (ساما).
