## 💡 المشكلة

خدمات التقسيط
(BNPL — تمارا، تابي)
انتشرت بين الشباب السعودي بسرعة تفوق وعيهم المالي. كل تطبيقات الميزانية الحالية تعمل **بأثر رجعي**: تخبرك أين صرفت بعد ما الفلوس راحت.

**وقفة يشتغل بالعكس** — يرصد إشارات الخطر ويصرخ **قبل** القسط اللي يكسر ظهرك.

## ✨ المميزات

| الميزة | القيمة |
|--------|--------|
| 🚨 **إنذار مبكر ثلاثي المستويات** | آمن → تحذير → خطر — يكتشفك في "تحذير" قبل الوصول لـ"خطر" |
| 🤖 **تحليل توليدي شخصي** | Groq llama-3.3-70b يحلل وضعك ويرد بالعربي مع خطوة عملية واحدة واضحة |
| 📈 **توقّعات مالية** | إسقاط مسارك 3 أشهر قادمة بناءً على نمطك الحالي |
| 🎛️ **محاكي "ماذا لو"** | سلايدرات حية + عدّاد خطر لحظي — جرّب أثر قسط جديد قبل توقيعه |
| 📷 **تصوير الفاتورة** | موديل رؤية يستخرج المتجر والمبلغ والتصنيف ويكشف لو الشراء بالتقسيط |
| 🏦 **محاكاة المصرفية المفتوحة** | تجميع 3 مصادر (بنك + محفظة + BNPL) بنقرة واحدة |
| 👤 **ملف تخصيصي** | العمر والدخل والأهداف تُحقن في برومبت الـ AI — نصيحة لك أنت، مو نصيحة عامة |
| 📚 **تعليم مالي تفاعلي** | اضغط أي مصطلح غامض → شرح فوري بالعربي |
| 📤 **تقرير قابل للمشاركة** + سجل تحليلات كامل |

## 🏗️ البنية

```
Flutter (Dart) — 11 شاشة، RTL، ثيم داكن، خط Cairo
│
├── التحليل: Groq API (llama-3.3-70b) + fallback محلي
│     └── لا يظهر خطأ خام للمستخدم أبداً — الاحتياطي يحلل محلياً
│
├── الرؤية: llama-4-scout (تحليل الفواتير → JSON منظم)
│
└── التخزين: shared_preferences (الملف، السجل) — كل شيء على الجهاز
```

**قرارات تصميم:**
- **حماية العرض الحي:** مهلات زمنية + fallback محلي — التطبيق يشتغل حتى لو انقطع النت لحظة العرض
- **prompt مقوّى:** system message + temperature منخفضة تمنع الموديل من الانحراف لحسابات خاطئة
- **الأسرار خارج git:** المفتاح في `secrets.dart` المستبعد — انظر التشغيل

## 🚀 التشغيل

```bash
flutter pub get

# انسخ قالب الأسرار وضع مفتاحك من console.groq.com
cp lib/services/secrets.dart.example lib/services/secrets.dart

flutter run
```
